分享你的镜像

使用Docker账号登录

如果你还没有账号，请在cloud.docker.com注册。注册好后，在命令行登录：

为你的镜像打上标签Tag

具体的语法：docker tag image username/repository:tag

  • image：哪一个镜像名；
  • username:用户名（一定要对上你的账户名）；
  • repository:库名；
  • tag ：版本号

例如：docker tag friendlyhello john/get-started:part2
